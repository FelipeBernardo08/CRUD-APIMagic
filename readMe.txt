Informações necessárias para usar a API

Comandos:
composer require slim/slim "^3.0"
composer require firebase/php-jwt
composer require vlucas/phpdotenv

Executar script para criação do banco de dados, esta presente nos arquivos da pasta do projeto

Configurar e criar o arquivo .env com as conexões do banco de dados local

Usar o login do cadastro admin para acessar as rotas, o mesmo foi criado na execução do script do banco de dados.

rota: http://localhost/APIMagic/ + endpoint desejado
